# Muni-NetworkMusic
## 基于uni-app框架开发的仿网易云音乐微信小程序

### 本项目目前实现了页面基本样式，登录，播放，视频，歌单，歌词，评论，底部控制栏，搜索，排行榜等功能
### 安装步骤
1、下载源码并导入VScode开发工具
2、npm install 安装依赖
3、npm run dev:mp-weixin 运行项目并在微信开发工具打开项目
4、在小程序开发管理中配置服务器域名
在request合法域名中填入 https://www.codeman.ink; 提交并保存即可
